# 🎬 Anime Tracker (C)

<p align="center">
  <img src="assets/banner.png" alt="Anime Tracker Banner" width="900">
</p>

Sistema de terminal desenvolvido em **linguagem C** para registrar, organizar e acompanhar o progresso de animes assistidos.

O programa funciona como um **rastreador pessoal de animes**, permitindo cadastrar séries, registrar episódios assistidos e acompanhar o progresso através de um **dashboard visual com barras de progresso**.

Os dados são armazenados em um **arquivo TXT**, funcionando como um pequeno banco de dados persistente.

---

# 🚀 Funcionalidades

- 📺 Cadastro de animes
- 📊 Barra de progresso visual
- 🎞 Controle de temporadas
- 🏷 Sistema de categorias  
  - Assistindo  
  - Completo  
  - Planejado  
  - Dropado  
- 🔎 Busca de anime por nome
- 🗑 Remoção de anime
- 🏆 Ranking automático por progresso
- 📂 Persistência de dados em arquivo `.txt`
- 🖥 Interface colorida no terminal

# 💾 Banco de dados

Os dados são armazenados em um arquivo **TXT** simples.

Formato:
nome|temporada|episodios_totais|episodios_assistidos|status
